# **App Name**: J-Auto Hub

## Core Features:

- Vehicle Showcase: Display available Japanese vehicles with details like make, model, year, engine, mileage, condition, images, and pricing.
- Admin Dashboard: Secure admin dashboard for managing vehicle listings and inventory.
- Incoming Vehicle Management: View and manage incoming vehicles before they arrive, tracking mileage and inspection status.
- Real-time Inventory Updates: Update vehicle availability status in real-time (incoming, available, sold).
- Sales History: View sales history with sale dates, buyer details, and final prices.
- Image Optimization: Automatically optimize images uploaded by admins to improve page load times. LLM Tool: If the images do not meet format standards, flag for further attention

## Style Guidelines:

- Primary color: Deep navy blue (#1A237E) to convey professionalism and trust.
- Background color: Light gray (#F5F5F5) to ensure readability and a clean aesthetic.
- Accent color: Muted gold (#BDBDBD) for highlights and key actions to symbolize quality and value.
- Body text font: 'Inter', sans-serif, for a clean, modern, neutral feel.
- Headline font: 'Space Grotesk', sans-serif, for short lines of text.
- Use clean, minimalist icons to represent vehicle features and statuses.
- Implement a grid-based layout for consistent and user-friendly navigation.
- Subtle transitions and animations to enhance user experience without being distracting.
